# GoRest Frontend Test Task

## Функциональность

- **Главная страница**: Поле для ввода access token с валидацией
- **Список пользователей**: Таблица с именем, фамилией, email; пагинация; клик для перехода к карточке пользователя
- **Список постов**: Таблица с ID и заголовком; пагинация; клик для перехода к карточке поста
- **Карточка пользователя**: Информация о пользователе и его посты
- **Карточка поста**: Полная информация о посте и все комментарии

## Технический стек

- React 19
- TypeScript
- Vite
- Consta UI Kit
- React Router DOM
- Zustand (управление состоянием)
- Axios (HTTP клиент)

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash

```

## Сборка для продакшена

```bash

```


## Структура проекта

```
src/
├── api/             # API сервисы для GoRest
├── components/      # Переиспользуемые компоненты
├── pages/          # Страницы приложения
├── store/         # Zustand хранилище
├── types/          # TypeScript типы
├── App.tsx         # Главный компонент
├── main.tsx        # Точка входа
└── style.css      # Глобальные стили
```

## Использование

1. Получите access token с https://gorest.co.in/
2. Введите токен в поле на главной странице
3. Переключайтесь между пользователями и постами с помощью табов
4. Используйте пагинацию для навигации по страницам
5. Кликайте по строкам для просмотра детальной информации
